# Self-Memory System (SMS)

**Summary**: Conway's conceptual framework in which the self and autobiographical memory are bidirectionally coupled. Memory is the database of an active self; the self modulates what memory encodes, retains, and retrieves.

**Sources**: Conway_2005.pdf

**Last updated**: 2026-05-27

---

## The framework

The SMS has two main components (source: Conway_2005.pdf):

1. The [working-self](/working-self): a hierarchy of active goals plus a conceptual self of self-images, scripts, and possible selves.
2. The [autobiographical-knowledge-base](/autobiographical-knowledge-base): a hierarchical store of life-story, themes, lifetime periods, and general events, with [episodic-memory](/episodic-memory) at its base.

When these two interlock during recall, a specific autobiographical memory is constructed. Either component can also operate alone in processing other than memory.

## The core claim

Memory is not playback. It is a goal-driven reconstruction. The self both depends on memory for self-knowledge and reshapes memory to remain coherent. Conway calls this the [coherence-vs-correspondence](/coherence-vs-correspondence) tradeoff.

A memory system that perfectly preserved experience would crush itself under storage demands and fail to track goal progress. A memory system that always edited to fit the current self would lose contact with reality. The SMS resolves this by routing recent goal-tracking through episodic memory (correspondence-biased) and long-term identity-relevant content through autobiographical knowledge (coherence-biased).

## How it constructs a memory

The retrieval cycle (see [retrieval-models](/retrieval-models)):

1. A cue activates part of the autobiographical knowledge base.
2. Evaluative control processes (the [working-self](/working-self)) elaborate the cue and decide whether to terminate or continue.
3. Generative search descends the hierarchy, looking for an episodic memory consistent with current goals.
4. When a stable activation pattern emerges across the relevant knowledge structures and an episodic image enters consciousness, [autonoetic-consciousness](/autonoetic-consciousness) signals that the result is a memory rather than a fantasy.

This takes 2-7 seconds for cued autobiographical recall, vs ~1 second for general factual recall, reflecting the construction time.

## Neuroanatomy

EEG and fMRI studies place SMS construction in fronto-temporal networks (control), temporal-pole anterior networks (conceptual autobiographical knowledge), and posterior occipital-temporal networks (episodic imagery). The medial temporal lobe and hippocampus bridge these. Conway proposes this as evidence for two memory systems with distinct evolutionary histories.
