# Conway 2005: Memory and the Self

**Summary**: Martin Conway's foundational paper introducing the Self-Memory System (SMS), a conceptual framework in which the self and autobiographical memory are reciprocally constructed. Memory is not video playback but a goal-driven construction process.

**Sources**: Conway_2005.pdf (Journal of Memory and Language 53, 594-628)

**Last updated**: 2026-05-27

---

## What the paper argues

The self is a complex set of active goals and self-images, collectively called the [working-self](/working-self). Long-term memory is its database. The two are bidirectionally coupled: autobiographical knowledge constrains what the self can plausibly be, but the working self also modulates what gets encoded, retained, and retrieved (source: Conway_2005.pdf).

A memory system that recorded experience faithfully would be overwhelmed and useless for goal-tracking. A memory system that distorted everything to fit current goals would lose contact with reality. Conway calls the compromise [coherence-vs-correspondence](/coherence-vs-correspondence): memory trades off between confirming current self-images and corresponding to actual events.

## Main components of the SMS

- The [working-self](/working-self): an active goal-hierarchy plus a conceptual-self of self-images, scripts, and possible-selves.
- The [autobiographical-knowledge-base](/autobiographical-knowledge-base): a hierarchical store from life story → themes → lifetime periods → general events.
- [episodic-memory](/episodic-memory): experience-near, image-based, short-time-slice records of goal-processing junctures. These are forgotten unless they integrate with the autobiographical knowledge base.

## Findings the paper covers

- The [reminiscence-bump](/reminiscence-bump): people aged 35+ disproportionately recall events from ages 10-30, the period when self-images form.
- Childhood amnesia: episodic memories before age 3-4 are largely inaccessible because the adult working self cannot interface with the infant goal-system.
- Memory inaccessibility as protection: working-self control processes can inhibit memories that threaten coherence (PTSD case studies, frontal-lobe confabulation, anosognosia).
- [generative-vs-direct-retrieval](/generative-vs-direct-retrieval): most autobiographical recall is iterative search, taking 2-7 seconds; some Proust-like retrieval is direct and rapid.
- Two memory systems hypothesis: a phylogenetically older episodic system (posterior temporo-occipital) and a newer conceptual-autobiographical system (anterior temporal-frontal). The hippocampus bridges them.

## Why this matters for the substrate question

Conway's framework is the psychological-level argument that memory is not storage. Memory is the construction process by which a self holds together across time. This is the load-bearing claim that [youssef-2026-psychology-and-ai-memory](/youssef-2026-psychology-and-ai-memory) later applies to AI agent architectures and that converges with [levin-2024-self-improvising-memory](/levin-2024-self-improvising-memory) at the substrate-agnostic level.
