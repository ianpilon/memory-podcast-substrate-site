# Memory as Substrate

A Karpathy-style LLM wiki on memory, identity, and the substrate question. Built from six sources synthesizing three converging arguments:

- **Psychology** ([Martin Conway](/conway-2005-memory-and-self)): the Self-Memory System. Memory and self are bidirectionally coupled; memory is reconstructed at every retrieval.
- **Biology** ([Michael Levin](/levin-2024-self-improvising-memory)): memory as agential cognitive glue. Active reinterpretation, no single substrate, scale-free across cells and lineages.
- **AI engineering** ([Robert Youssef](/youssef-2026-psychology-and-ai-memory)): the paradigm shift from retrieval to identity construction. Five principles current AI memory lacks.
