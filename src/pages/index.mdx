# Memory as Construction

A Karpathy-style LLM wiki on memory, identity, and the construction question. Built from seven sources synthesizing five converging arguments. See [On the Rename](/on-naming) for the arc from "substrate" to "navigation" to "construction".

- **Psychology** ([Martin Conway](/conway-2005-memory-and-self)): the Self-Memory System. Memory and self are bidirectionally coupled; memory is reconstructed at every retrieval.
- **Biology** ([Michael Levin](/levin-2024-self-improvising-memory)): memory as agential cognitive glue. Active reinterpretation, no single substrate, scale-free across cells and lineages.
- **AI engineering, principles** ([Robert Youssef](/youssef-2026-psychology-and-ai-memory)): the paradigm shift from retrieval to identity construction. Five principles current AI memory lacks.
- **AI engineering, measured** ([NapMem](/source-napmem)): memory as a structured action space the agent navigates, not passively retrieved context.
- **AI engineering, intervention** ([Proactive Memory Agent](/source-proactive-memory)): memory as intervention. A separate memory agent decides whether to inject a reminder or stay silent.
