# Memories as Agents

**Summary**: Levin's proposal that memories are not passive data but active patterns with a degree of agency. They cooperate and compete in their host cognitive systems to shape the mind that holds them. "Thoughts are thinkers," in William James's phrase.

**Sources**: Self-Improvising Memory.pdf, self-improvising-memories.pdf

**Last updated**: 2026-05-27

---

## The claim

Levin proposes giving up the binary dichotomy between computational machinery and passive data. Memories, especially non-local ones, may actively facilitate their own transformation and remapping in cognitively excitable media. Instead of agents holding memories, memories are partly **agential structures that shape the agents that hold them** (source: Self-Improvising Memory.pdf).

This is a strong claim. Levin acknowledges it might prove too strong, and a moderate hybrid version may be the right answer.

## The continuum

Rather than a sharp data-vs-mind boundary, Levin proposes a continuum of agency:

- Fleeting thoughts.
- Persistent or intrusive thoughts.
- Metastable entities (tulpas, dissociative alters as more durable patterns).
- Conventional full personalities (minds) able to generate all the prior members.

In this view, an agent is "just a metastable pattern in an excitable medium, a temporarily persistent pattern." That is also what humans are: temporarily persistent, autocatalytic, dissipative patterns that self-reify our boundaries by active inference.

## Engineering analogue

Benjamin L, in the comment thread of [levin-2024-blog-supplement](/levin-2024-blog-supplement), offers a useful analogy. Lisa Feldman Barrett's theory of [constructed-emotion-barrett](/constructed-emotion-barrett) treats concepts as "competing neural patterns that strive to be the best available prediction for matching incoming data." In that view, "thoughts are thinkers" is quite literally true: thoughts compete to be implemented, and the winner becomes the current experience.

In machine learning terms, this is similar to the dynamics inside a transformer: many candidate continuations compete, the highest-probability one is sampled. But Levin's claim is stronger: the **engram itself** has minimal agency, not just the inference process.

## Why this matters

[youssef-2026-psychology-and-ai-memory](/youssef-2026-psychology-and-ai-memory) argues that current AI memory is built on the database metaphor: store, retrieve, done. If Levin is right, that metaphor is fundamentally inadequate. An agent's memory should be **active patterns that shape the agent's responses**, not passive content the agent looks up.

This is also the deepest version of the [memory-as-substrate-thesis](/memory-as-substrate-thesis). Substrate-agnostic, scale-free, agential memory is the core architectural commitment.
