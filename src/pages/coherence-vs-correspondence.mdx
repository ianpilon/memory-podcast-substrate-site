# Coherence vs Correspondence

**Summary**: Conway's framing of the core tradeoff in human memory. Coherence is consistency with current goals, self-images, and beliefs. Correspondence is fidelity to the actual events. The healthy memory system balances the two; pathologies of memory are failures of that balance.

**Sources**: Conway_2005.pdf

**Last updated**: 2026-05-27

---

## The distinction

The terms come from Russell's distinction in philosophy of mind: coherence theories of truth depend on agreement among propositions, correspondence theories on agreement with facts. Conway, Meares, and Standart (2004) applied this to memory (source: Conway_2005.pdf).

- A pure-correspondence memory system would faithfully record everything. This is what amnesic patient Clive Wearing's behaviour reveals as the cost: without integration into goal-tracking, the system collapses (see [clive-wearing](/clive-wearing)).
- A pure-coherence memory system would always produce a self-consistent story. This is what frontal-damage confabulation reveals: false but self-consistent memories that obliterate the connection to reality.

The healthy system is "adaptively coherent": there is an optimum retention for any experience that maximizes fitness and survival.

## Where each dominates

- **Recent memory** is biased toward correspondence. You need to know whether you actually locked the door, because acting on a false memory is costly.
- **Long-term memory** is biased toward coherence. You need a self-consistent story to operate effectively across years.

Episodic memory does most of the correspondence work. The autobiographical knowledge base does most of the coherence work. The [working-self](/working-self) modulates both.

## Coherence as protective distortion

Conway documents many cases where memory distortion serves coherence. In PTSD survivors, intrusive memories often contain errors that protect the self from major change (a "different person" experience after trauma). Cognitive behavioural therapy works in part by reinstating correspondence and disabling coherence-driven distortion (see [ptsd-and-memory-distortion](/ptsd-and-memory-distortion)).

A 9/11 witness with PTSD recalled herself watching from above the scene with "no noise," in a peaceful image. The treatment included reinstating field perspective and the sounds of the moment, after which PTSD symptoms lessened. The original memory was coherence-protective (it kept the witness emotionally distant) but interfered with recovery.

## Why this matters

This is the deepest psychological-level alignment with [levin-2024-self-improvising-memory](/levin-2024-self-improvising-memory): biology gives up on fidelity and commits to salience. Conway's framework gives the conditions under which fidelity still matters (recent goal-tracking) and the conditions under which it does not (long-term identity).

Both authors are arguing, from different disciplines, that memory's job is to keep an agent operating across time, not to archive the past. See [memory-as-navigation-thesis](/memory-as-navigation-thesis).
