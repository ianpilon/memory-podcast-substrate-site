# Levin 2024: Self-Improvising Memory

**Summary**: Michael Levin's Entropy paper proposing that memory is not data storage but a dynamic agent-based reinterpretation process operating across all biological scales. Memories are messages between past and future Selves, and their reliability comes from creative re-interpretation, not faithful playback.

**Sources**: Self-Improvising Memory.pdf (Entropy 2024, 26, 481)

**Last updated**: 2026-05-27

---

## What the paper argues

Biology faces a paradox: if a species fails to change it dies, but if it changes too much the past Self ceases to exist. Levin's resolution is that biology gave up on fidelity and committed to salience. Memory does not preserve low-level details. It preserves whatever pattern remains useful to a future, possibly very different, agent (source: Self-Improvising Memory.pdf).

This view applies at every scale Levin studies: cells, tissues, organisms, swarms, ecosystems, even evolutionary lineages. The same architecture explains how a caterpillar's training survives metamorphosis into a butterfly even though the butterfly's body, environment, and goals are completely different.

## Core moves

- [mnemonic-improvisation](/mnemonic-improvisation): the dynamic ability to rewrite and remap memories onto new media and contexts.
- [memories-as-agents](/memories-as-agents): engrams are not passive data but agential prompts that actively shape the cognitive system that holds them.
- [bowtie-architecture](/bowtie-architecture): information compresses to a low-bandwidth hub, then must be creatively re-inflated. The same shape appears in autoencoders, language, prices, and DNA.
- [polycomputing](/polycomputing): any physical process can be read as performing different computations by different observers. Biology exploits this for hacking and reuse.
- [self-as-process](/self-as-process): the Self is a process, not a thing. Snapshot-Selves ([selflets](/selflets)) are low-dimensional projections of a deeper continuous flow.
- [confabulation-as-feature](/confabulation-as-feature): confabulation, the active filling-in of gaps and rewriting of past data, is the central feature of cognition, not a bug.
- [no-single-substrate](/no-single-substrate): memory is not localized to synapses or any one medium. Biological structure at all levels can serve as an interpretable scratchpad.

## Key examples

- Caterpillar-to-butterfly memory transfer (the body that learned is largely dissolved, yet useful associations survive in a radically different body and environment).
- Planarian regeneration of trained behaviour from tail fragments.
- Memory transfer between animals via brain extracts.
- Aplysia learning transferred through RNA injection.
- Tadpoles with eyes on their tails learning to see.
- Salamander tails grafted onto flanks that remodel into limbs.

## Why this matters for the substrate question

Levin's framework is the biological-level argument that memory is substrate-agnostic. Combined with [conway-2005-memory-and-self](/conway-2005-memory-and-self)'s psychological argument that memory constructs the self, this implies that the right way to build artificial memory is not to copy hippocampi or synapses, but to build any medium that can sustain agential reinterpretation across time. See [memory-as-navigation-thesis](/memory-as-navigation-thesis).
